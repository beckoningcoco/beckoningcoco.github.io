什么是反射？

  • 反射就是通过加载类，对类中各个成分（成员变量，方法，构造器）进行操作。

反射的学习内容

  1. 反射第一步，获取类的字节码文件：Class对象
  2. 获取类的构造器：Constructor 对象

  4. 获取类的成员方法： Method对象
